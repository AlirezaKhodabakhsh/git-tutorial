## Linux commands


### Current directory
```bash
pwd
```
> /d/GitHub/


### Change directory
`cd <desired directory>`
```bash
cd /d/GitHub/
```


#### Navigate back folder
`cd ..`
```bash
cd ..
```
>cd /d/


#### Navigate forward folder
`cd <desired folder name>`
```bash
cd GitHub
```
>cd /d/GitHub/

### Make new folder
`mkdir <desired new folder name>`
```bash
mkdir gittutorial
```

### List current folder contents
`ls`


### Make new file
`touch <file name>`
```bash
touch index.py
```
